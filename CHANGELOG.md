# 修改记录

## 2026-05-06

### 新增功能

- **目标岗位列表页** — 管理目标岗位（JD），支持手动输入或截图上传自动 OCR 识别，AI 从经历池挑选最匹配经历创建定制简历
- **经验池自动去重** — 模糊匹配公司名+职位，识别"大体相同"的经历自动合并（如 "Google Inc" ≈ "Google"，"字节跳动" ≈ "字节跳动有限公司"）
- **JD 截图识别** — 支持多张图片上传，Tesseract.js 浏览器端 OCR 提取文字，AI 整理格式，无需多模态模型
- **阿里云百炼预设** — 设置页新增 DashScope API 预设，端点 `dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
- **导航入口** — 主导航新增"目标岗位"链接
- **面试页目标岗位选择** — 可从已保存的目标岗位列表一键填充公司/岗位/JD

### 优化改进

- **卡片标题内联编辑** — 首页工作台、简历列表页、简历编辑页、面试历史记录卡片 blob 头部均支持点击 PenLine 图标编辑标题
- **API 连接测试错误提示** — 设置页显示具体错误信息，而非笼统的"连接失败"
- **API 代理超时** — 从 120s 延长到 300s，解决创建定制简历时请求超时
- **面试 session 卡片** — 支持用户编辑面试标题

### Bug 修复

- **React key 警告** — AI 生成的 skills 缺少 id，添加 `uuidv4()` 兜底
- **API 端点错误** — 修正百炼 API URL（之前用户手动填写了错误的 `api.bailian.aliyuncs.com`）
